# 📊 A股分析系统

实时分析A股股票，辅助日内做T和短期操作决策。

## 功能

- 📈 **实时行情**：获取A股实时价格和涨跌幅
- 💹 **技术指标**：MA、MACD、KDJ、RSI等常用指标
- 📉 **资金流向**：主力资金、超大单、大单等资金分析
- 🎯 **做T辅助**：盘中高低点提示

## 快速开始

1. 打开页面
2. 输入股票代码
3. 查看实时分析和指标

## 技术栈

- 前端: 原生 HTML5 / CSS3 / JavaScript
- 部署: GitHub Pages
- 东财/新浪金融数据API
